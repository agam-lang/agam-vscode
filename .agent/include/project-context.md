# Project Context

## What This Repository Is

- Repository: `agam-lang/agam-vscode`
- Purpose: Provide first-class VS Code support for Agam development including syntax highlighting, LSP integration, snippets, debugging, and code actions.
- Technology: TypeScript + VS Code Extension API

## Organization Context

- This is part of the `agam-lang` ecosystem.
- The core Agam compiler lives at `agam-lang/agam`.
- The standard library lives at `agam-lang/std`.
- The scientific platform lives at `agam-lang/agamlab`.

## Key Areas

- Syntax Highlighting
- LSP Client
- Snippets
- Debugging
- Code Actions
- Themes
